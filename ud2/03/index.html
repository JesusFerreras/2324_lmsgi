Plantilla de formulario

Tipos de input

Tipo text
Registra una línea de texto
Tipo radio
Registra una elección única entre varias opciones
Tipo checkbox
Registra una o varias elecciones
Tipo email
Registra una dirección de correo electrónico, algunos navegadores la validan automáticamente
Tipo tel
Registra un número de teléfono
Tipo date
Registra una fecha
Tipo time
Registra una hora
Tipo week
Registra un número de semana y un año
Tipo month
Registra un mes y año
Tipo range
Registra un valor numérico aproximado mediante un deslizador
Lista desplegable
Lista de opciones predefinidas
Textarea
Puede registrar varias líneas de texto, su tamaño es ajustable
Tipo imagen
Envia los datos introducidos, el botón tiene una imagen en lugar de texto
Tipo hidden
Registra datos que no son visibles ni variables para el usuario
Tipo search
Registra una línea de texto, puede ser usada para realizar búsquedas
Tipo url
Registra una url, algunos navegadores la validan automáticamente
Tipo password
Registra una línea de texto pero no muestra los caracteres que la componen
Tipo number
Registra un número
Tipo file
Registra un archivo del equipo, incluye el botón "Seleccionar archivo"
Tipo color
Registra un color
Tipo submit
Envia los datos introducidos
Tipo reset
Restablece los datos introducidos

Distintos atributos

Atributo Descripción
Checked La opción aparecerá marcada por defecto
Disabled Indica que el campo debe ser deshabilitado
Max Indica el valor máximo que puede tener una variable
Maxlength Indica la cantidad de caracteres máximos que puede tener una variable
Min Indica el valor mínimo que puede tener una variable
Minlength Indica la cantidad de caracteres mínimos que puede tener una variable
Pattern Indica cómo debe ser la expresión regular con la que se compararán los datos
Readonly Indica que un campo no puede ser cambiado
Required Indica que el campo debe ser rellenado
Size Indica la cantidad de caracteres que debe tener una variable
Step Indica el intervalo por el cual debe cambiar una variable
Value Indica el valor por defecto que tendrá una variable
